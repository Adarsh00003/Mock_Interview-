# Mock Interview Platform 🎯

An AI-powered mock interview application where users can upload resumes, analyze their skills, and practice interview questions interactively.

---

## 🚀 Features
- Secure user authentication (Firebase + JWT)
- Resume upload & PDF parsing
- AI-driven resume analysis (role, experience, projects, skills)
- Mock interview simulation
- Frontend deployed on **Vercel**
- Backend deployed on **Render**

---

## 🛠️ Tech Stack
- **Frontend:** React (Vite), Axios, Bootstrap  
- **Backend:** Node.js, Express.js, Multer, JWT  
- **Database:** MongoDB (Mongoose)  
- **AI Integration:** Gemini / OpenAI  
- **Auth:** Firebase Authentication
